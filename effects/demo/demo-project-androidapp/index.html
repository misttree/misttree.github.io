Android APP开发：华容道游戏开发设计

1 项目描述

1.1 项目背景

该项目基于南开大学软件学院智能移动开发课程讲授内容进行设计,使用Android Studio平台进行开发, 位于Android端的手机游戏APP：三国华容道

1.2 操作环境

  • 编程语言：Java
  • 编程平台：Android Studio
  • 服务器：待定
  • 数据库：MySQL

1.3 设计思路

基于Android运行平台的开发,使用Android Studio可以进行拖拽式的开发模式,目前的基本思路为使用GridLayout来进行游戏界面的处理 ,在其中添加事件的响应处理,从而达到用户移动的效果
